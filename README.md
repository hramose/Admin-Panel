# Admin Panel for Laravel 5 #

## ***Install:*** ##

Composer:

```
#!json

"cinject/admin-panel": "dev-master"
```

Provider:

```
#!php

'Cinject\AdminPanel\Providers\AdminPanelServiceProvider',
```

Publish resource:

```
#!bash
php artisan vendor:publish --provider='Cinject\AdminPanel\Providers\AdminPanelServiceProvider'
```
